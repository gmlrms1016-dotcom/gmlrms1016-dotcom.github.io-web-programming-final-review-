## 주요 기능

- **과목 선택 허브** — `index.html` 첫 화면에서 5과목으로 이동
- **인앱 파이썬 실행·채점** — `pyrun.js`(Pyodide)로 브라우저 안에서 코드를 직접 실행하고 정답/오답 채점 (프로그래밍방법론·파이썬프로그래밍)
- **틀린 문제 모아 복사** — `review.js` 공통 기능으로 객관식·괄호채우기 오답을 한 번에 수집 (전 과목)
- **선명한 순서도** — 프로그래밍방법론의 Flowgorithm 캡처를 화질 손상 없는 SVG로 재작성

## 파일 구성

- `index.html` — 과목 선택 허브(첫 화면)
- `웹프로그래밍기초.html` · `컴퓨터공학기초.html` · `SW개발도구활용.html` · `프로그래밍방법론.html` · `파이썬프로그래밍.html` — 과목별 정리 페이지
- `style.css` — 공용 스타일시트(초록 테마)
- `review.js` — 틀린 문제 모아 복사 (전 과목 공통)
- `pyrun.js` — Pyodide 기반 인앱 파이썬 실행·채점
- `grader_pm.py` — 프로그래밍방법론 채점 로직 / `grader.py` — 파이썬프로그래밍 채점 로직
- `pm-img/` — 프로그래밍방법론 과제 캡처 / `sw-img/` — SW개발도구활용 이미지
