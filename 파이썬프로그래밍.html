01. 점수 → 학점 변환 (match-case)grade_학번

📄 문제 : match-case 구문을 활용하여 입력받은 점수를 학점으로 반환하는 프로그램을 완성하시오. 함수의 매개변수는 int로 타입 힌트를 준다.

▸ 90이상 100이하 A+ · 80이상 90미만 A · 70이상 80미만 B+ · 60이상 70미만 B · 50이상 60미만 C+ · 40이상 50미만 C · 나머지 F

출력 결과

점수를 입력하세요: 90
당신의 학점은 A+ 입니다.
점수를 입력하세요: 200
잘못된 점수입니다. 0에서 100 사이의 값을 입력해주세요.
점수를 입력하세요: A
유효하지 않은 입력입니다. 숫자를 입력해주세요.

핵심 개념

개념설명
match-case파이썬 3.10+ 분기 구문. case _ if 조건: 형태의 가드(guard)로 범위 조건을 검사
타입 힌트def get_grade(score: int): 처럼 매개변수 자료형을 표기(강제는 아님)
입력 검증score.isdigit()로 숫자인지 확인 후 int() 변환, 0 <= score <= 100 범위 검사

풀이 (정답 코드)

def get_grade(score: int):              # 점수(int 타입 힌트)를 받아 학점 문자열을 반환
    match score:                        # score 값으로 분기 (match-case, 파이썬 3.10+)
        case _ if 90 <= score <= 100:   # 90~100 → A+
            return "A+"
        case _ if 80 <= score < 90:     # 80~89 → A
            return "A"
        case _ if 70 <= score < 80:     # 70~79 → B+
            return "B+"
        case _ if 60 <= score < 70:     # 60~69 → B
            return "B"
        case _ if 50 <= score < 60:     # 50~59 → C+
            return "C+"
        case _ if 40 <= score < 50:     # 40~49 → C
            return "C"
        case _:                         # 나머지(40 미만) → F
            return "F"


def main():
    while True:
        score = input("점수를 입력하세요: ")
        if score.isdigit():                       # 숫자로만 이뤄졌는지 검증
            score = int(score)
            if 0 <= score <= 100:
                print(f"당신의 학점은 {get_grade(score)} 입니다.")
            else:
                print("잘못된 점수입니다. 0에서 100 사이의 값을 입력해주세요.")
        else:
            print("유효하지 않은 입력입니다. 숫자를 입력해주세요.")


if __name__ == "__main__":      # 직접 실행할 때만 (import 시 실행 안 됨)
    main()

02. 문자열 각 단어 길이 추출 (map · zip)lengths_학번

📄 문제 : map(len, words)를 활용해 입력 문자열에서 각 단어의 길이를 추출하는 프로그램을 완성하시오.

출력 결과

문자열을 입력하세요: Your submitted manuscript is currently under review
각 단어의 길이:
Your: 4글자
submitted: 9글자
manuscript: 10글자
is: 2글자
currently: 9글자
under: 5글자
review: 6글자

핵심 개념

개념설명
split()문자열을 공백 기준으로 나눠 단어 리스트 생성 (예: "a b c".split() → ['a','b','c'])
map(함수, 반복형)각 원소에 함수를 일괄 적용. map(len, words) = 각 단어 길이. list()로 리스트화
zip()여러 리스트를 같은 위치끼리 묶음. for word, length in zip(단어들, 길이들) 로 짝지어 출력

풀이 (정답 코드)

def calculate_word_lengths(text):       # 문자열을 받아 각 단어 길이 리스트를 반환
    words = text.split()                # split(): 공백 기준으로 단어 분리
    return list(map(len, words))        # map(len, words): 각 단어 길이를 리스트로


input_text = input("문자열을 입력하세요: ")
word_lengths = calculate_word_lengths(input_text)   # 각 단어 길이 리스트 생성

print("각 단어의 길이:")
for word, length in zip(input_text.split(), word_lengths):  # 단어·길이를 짝지어 꺼냄
    print(f"{word}: {length}글자")                  # "단어: N글자" 형식 출력

03. turtle로 별 6개 임의 위치·색칠 (random)turtlestar_학번

📄 문제 : 모듈 turtle로 임의의 위치에 별 6개를 그리고, 리스트에 지정한 색으로 각각 다르게 칠하는 프로그램을 작성하시오.

▸ 색상 color = [red, orange, yellow, green, blue, magenta] · 모양 거북이 · 창 600×600 · 속도 3 · 한 변 100픽셀

핵심 개념

개념설명
random.randint(a, b)a~b 사이 무작위 정수. random.randint(-200, 200)로 x·y 좌표 생성
펜 제어 pu()/pd()pu()(=penup) 펜 올려 이동(자국 X), pd()(=pendown) 펜 내려 그리기
채우기 fillcolor/begin_fill/end_fillfillcolor(색)로 색 지정 후 begin_fill()…end_fill() 사이 도형을 색칠
별 모양한 변 100픽셀, forward(100) 후 right(144)를 5번 반복하면 5각 별 ★

풀이 (정답 코드)

import turtle as t
import random

color = ["red", "orange", "yellow", "green", "blue", "magenta"]  # 색상 6개

t.shape("turtle")          # 모양: 거북이
t.setup(600, 600)          # 그래픽 창 600x600
t.speed(3)                 # 속도 3

for i in range(6):                       # 별 6개 반복
    x = random.randint(-200, 200)        # 무작위 x좌표 (-200~200)
    y = random.randint(-200, 200)        # 무작위 y좌표
    t.pu()                               # 펜 올리기(이동 자국 X)
    t.goto(x, y)                         # 임의 위치로 이동
    t.pd()                               # 펜 내리기(그리기 시작)

    t.fillcolor(color[i])                # i번째 색을 채우기 색으로
    t.begin_fill()                       # 채우기 시작
    for _ in range(5):                   # 별(★) 한 변씩 5번
        t.forward(100)                   # 한 변 100픽셀 전진
        t.right(144)                     # 144도 회전 → 별 모양
    t.end_fill()                         # 채우기 끝

t.done()                                 # 창 유지
💡 시험지는 별(★)이라 위는 range(5)+right(144)로 그렸습니다. 만약 사각형이면 안쪽 반복을 for _ in range(4): forward(100); left(90) 로 바꾸면 됩니다. 그릴 개수를 입력받으려면 count=int(input()) + 1~6 범위 검증 후 range(count).

04. 사물함 번호 배정 (try/except · random.sample)locker_학번

📄 문제 : n명의 학생에게 두 자리 자연수 번호를 갖는 사물함을 배정하는 프로그램을 작성하시오. (학생 수 1~10, x 입력 시 종료)

▸ student_count() try/except 예외처리·반복 입력 · locker_num() random.sample 중복없이 배정 · main() import 시 실행 안 됨

출력 결과

사물함 배정을 시작합니다.
학생수를 입력하세요: 150
범위를 벗어났습니다. 올바른 값을 입력하세요.
학생수를 입력하세요: a
숫자가 아닙니다. 올바른 값을 입력하세요.
학생수를 입력하세요: 10
10명에게 배정된 사물함 번호는 다음과 같습니다: [12, 24, 33, 86, 25, 27, 56, 72, 45, 90]

핵심 개념

개념설명
try / except오류가 날 수 있는 코드를 try:에, 오류 처리(숫자 변환 실패 등)를 except Exception as e:에 작성
random.sample(범위, k)중복 없이 k개 무작위 추출. range(10, 100) = 두 자리 자연수 10~99
if __name__ == "__main__"파일을 직접 실행할 때만 main() 호출, import 되면 실행 안 됨

풀이 (정답 코드)

import random

def student_count():                     # 학생 수를 입력받아 검증 후 반환
    while True:
        try:                             # 숫자 변환 오류 대비 예외 처리
            a = input("학생수를 입력하세요: ")
            if a == "x" or a == "X":     # x 입력 시 종료
                print("프로그램 종료.")
                break                    # while 탈출 → None 반환
            input_count = int(a)         # 정수 변환(실패 시 except로)
            if input_count < 1 or input_count > 10:
                print("범위를 벗어났습니다. 올바른 값을 입력하세요.")
            else:
                return input_count       # 1~10 유효값 반환
        except Exception as e:           # 숫자가 아닌 입력 등 모든 예외
            print("숫자가 아닙니다. 올바른 값을 입력하세요.")


def locker_num(count):                   # 사물함 번호 배정
    return random.sample(range(10, 100), count)   # 10~99 중 중복없이 count개


def main():
    print("사물함 배정을 시작합니다.")
    count = student_count()
    if count == None:                    # x로 종료한 경우
        exit()
    lockers = locker_num(count)
    print(f"{count}명에게 배정된 사물함 번호는 다음과 같습니다: {lockers}")


if __name__ == "__main__":      # import 시 실행되지 않도록
    main()
