01. 점수 → 학점 변환 (match-case)grade_학번

📄 문제 : match-case 구문으로 점수를 학점으로 반환하는 get_grade(score)를 작성하세요. main()은 이미 주어져 있습니다.

▸ 90이상 100이하 A+ · 80이상 90미만 A · 70이상 80미만 B+ · 60이상 70미만 B · 50이상 60미만 C+ · 40이상 50미만 C · 나머지 F

출력 결과

점수를 입력하세요: 90
당신의 학점은 A+ 입니다.
점수를 입력하세요: 200
잘못된 점수입니다. 0에서 100 사이의 값을 입력해주세요.
점수를 입력하세요: A
유효하지 않은 입력입니다. 숫자를 입력해주세요.

핵심 개념

개념설명
match-case파이썬 3.10+ 분기. case _ if 90<=score<=100: 가드 또는 str(score).zfill(2)[0] 맨 앞자리 분기
타입 힌트def get_grade(score: int):
채점 방식get_grade(점수)를 여러 점수로 호출해 학점이 맞는지 자동 검사

✍️ 코드 작성 & 실행


  

  
💡 정답 코드 보기
    
def get_grade(score: int):                 # 점수(int 타입 힌트)를 받아 학점 문자열 반환
    match str(score).zfill(2)[0]:          # 점수를 2자리 문자열로 만든 뒤 '맨 앞 자리'로 분기
        case "9":
            return "A+"
        case "8":
            return "A"
        case "7":
            return "B+"
        case "6":
            return "B"
        case "5":
            return "C+"
        case "4":
            return "C"
        case "1" if score == 100:
            return "A+"
        case _:
            return "F"


def main():
    while True:
        score = input("점수를 입력하세요: ")
        if score.isdigit():
            score = int(score)
            if 0 <= score <= 100:
                print(f"당신의 학점은 {get_grade(score)} 입니다.")
            else:
                print("잘못된 점수입니다. 0에서 100 사이의 값을 입력해주세요.")
        else:
            print("유효하지 않은 입력입니다. 숫자를 입력해주세요.")


if __name__ == "__main__":
    main()

    
채점은 get_grade 함수만 호출합니다(입력 루프는 실행 안 함). 0~100 전 구간을 검사해요.

  






  
02. 문자열 각 단어 길이 추출 (map · zip)lengths_학번

  
📄 문제 : map(len, words)로 입력 문자열에서 각 단어의 길이를 추출해 출력하세요.

  
출력 결과

  
문자열을 입력하세요: Your submitted manuscript is currently under review
각 단어의 길이:
Your: 4글자
submitted: 9글자
manuscript: 10글자
is: 2글자
currently: 9글자
under: 5글자
review: 6글자

  
핵심 개념

  
개념설명

    
split()문자열을 공백 기준으로 나눠 단어 리스트 생성

    
map(len, words)각 단어 길이 계산 → list()로 변환

    
zip()단어와 길이를 짝지어 "단어: N글자" 출력

  

  
✍️ 코드 작성 & 실행

  

    
    

    

    

  

  
💡 정답 코드 보기
    
def calculate_word_lengths(text):
    words = text.split()
    return list(map(len, words))


input_text = input("문자열을 입력하세요: ")
word_lengths = calculate_word_lengths(input_text)

print("각 단어의 길이:")
for word, length in zip(input_text.split(), word_lengths):
    print(f"{word}: {length}글자")

    
채점은 위 예시 문자열을 입력으로 넣고 출력이 정답과 같은지 비교합니다.

  






  
03. turtle로 사각형 임의 위치·색칠 (random)turtlestar_학번

  
📄 문제 : turtle로 임의 위치에 사각형을 그리고 각각 다른 색으로 칠하세요. 개수(1~6)를 입력받습니다.

  
▸ 색상 [red, orange, yellow, green, blue, magenta] · 거북이 · 600×600 · 속도 3 · 한 변 100

  
핵심 개념

  
개념설명

    
random.randint(-200,200)x·y 무작위 좌표

    
pu()/pd()펜 올림(이동)/내림(그리기)

    
fillcolor + begin_fill/end_fill색 지정 후 그 사이 도형을 색칠

    
사각형forward(100) 후 left(90) ×4

  

  
✍️ 코드 작성 & 실행

  

    
    

    

    

  

  
💡 정답 코드 보기
    
import turtle as t
import random

color = ["red", "orange", "yellow", "green", "blue", "magenta"]

while True:
    count = int(input("몇개?(1~6범위 입력)"))
    if 1 <= count <= 6:
        print("정상범위 입니다. 프로그램을 실행합니다.")
        break
    else:
        print("비정상범위 입니다. 다시 입력 부탁드립니다.")
        continue

t.shape("turtle")
t.setup(600, 600)
t.speed(3)

for i in range(count):
    x = random.randint(-200, 200)
    y = random.randint(-200, 200)
    t.pu()
    t.goto(x, y)
    t.pd()

    t.fillcolor(color[i])
    t.begin_fill()
    for _ in range(4):
        t.forward(100)
        t.left(90)
    t.end_fill()

t.done()

    
⚠️ 브라우저에선 그래픽 창을 못 띄웁니다. 가상 실행으로 '도형 개수·forward(100)·색칠' 동작을 검사해 채점합니다. (시험지는 ‘별’이지만 답지가 사각형이라 사각형 기준)

  






  
04. 사물함 번호 배정 (try/except · random.sample)locker_학번

  
📄 문제 : n명에게 두 자리 자연수 사물함 번호를 배정하세요. (1~10명, x 입력 시 종료)

  
▸ student_count() try/except 검증 · locker_num() random.sample · main() import 시 실행 안 됨

  
출력 결과

  
사물함 배정을 시작합니다.
학생수를 입력하세요: 150
범위를 벗어났습니다. 올바른 값을 입력하세요.
학생수를 입력하세요: a
숫자가 아닙니다. 올바른 값을 입력하세요.
학생수를 입력하세요: 10
10명에게 배정된 사물함 번호는 다음과 같습니다: [12, 24, 33, 86, 25, 27, 56, 72, 45, 90]

  
핵심 개념

  
개념설명

    
try / except숫자 변환 실패 등 예외 처리

    
random.sample(range(10,100), k)두 자리(10~99) k개 중복 없이

    
if __name__ == "__main__"직접 실행할 때만 main() 호출

  

  
✍️ 코드 작성 & 실행

  

    
    

    

    

  

  
💡 정답 코드 보기
    
import random

def student_count():
    while True:
        try:
            a = input("학생수를 입력하세요: ")
            if a == "x" or a == "X":
                print("프로그램 종료.")
                break
            input_count = int(a)
            if input_count < 1 or input_count > 10:
                print("범위를 벗어났습니다. 올바른 값을 입력하세요.")
            else:
                return input_count
        except Exception as e:
            print("숫자가 아닙니다. 올바른 값을 입력하세요.")


def locker_num(count):
    return random.sample(range(10, 100), count)


def main():
    print("사물함 배정을 시작합니다.")
    count = student_count()
    if count == None:
        exit()
    lockers = locker_num(count)
    print(f"{count}명에게 배정된 사물함 번호는 다음과 같습니다: {lockers}")


if __name__ == "__main__":
    main()

    
채점은 학생 수 10을 입력으로 넣고, 두 자리 번호 10개가 중복 없이 출력되는지 확인합니다.
