1) 순서도 기호

타원 = 시작/끝 평행사변형 = 입력/출력 직사각형 = 처리(대입·계산) 마름모 = 조건(판단) → 화살표 = 실행 흐름
기호의미파이썬 예
타원(시작/끝)프로그램의 시작과 종료코드의 처음·끝
평행사변형(입출력)값을 입력받거나 출력input() / print()
직사각형(처리)변수에 값 대입·계산add = a + b
마름모(조건)참/거짓으로 흐름 분기if ... else ...

2) 3대 제어구조

구조순서도 모양파이썬
순차위 → 아래 일렬한 줄씩 차례대로 실행
선택마름모에서 참/거짓 갈래if / elif / else
반복되돌아가는(루프) 화살표for / while

3) 내가 제출한 Flowgorithm 순서도 (실제 과제 캡처)

9~14주차 과제로 직접 그린 순서도입니다. 변환 규칙 → 마름모는 if/elif/else, 육각형(반복)은 for/while, 서브프로그램은 def, 반환값은 return. (이미지를 클릭하면 크게 볼 수 있어요)

① 선택구조 — 마름모(조건)로 분기

② 반복구조 — 육각형(루프)

③ 함수 — 서브프로그램 + 반환값

④ 클래스 — 순서도엔 없음, 파이썬에서만 (★ 시험 중요)

4) 한눈에 보는 예제 — 가위바위보 순서도(클린 재작성)

아래 순서도가 곧 2번 문제(judge)의 설계도입니다. 마름모(조건) → if, 중첩 마름모 → 중첩 if로 그대로 옮기면 됩니다.

💡 위 순서도를 파이썬으로 옮기면 (2번 정답과 동일) 
def judge(player, computer):
    if player == computer:          # 마름모1
        return "비김"
    else:
        if (player == "가위" and computer == "보") or \
           (player == "바위" and computer == "가위") or \
           (player == "보" and computer == "바위"):   # 마름모2
            return "이김"
        else:
            return "짐"

01. 윤년 판별 (선택구조)13주 · 선택

📄 문제 : 연도를 받아 윤년이면 "윤년", 평년이면 "평년"을 반환하는 leap(year)를 작성하세요.

▸ 윤년 규칙 : 4의 배수 그리고 100의 배수가 아님 또는 400의 배수

핵심 개념

개념설명
나머지 연산 %year % 4 == 0 → 4로 나누어떨어짐
and / or 우선순위and가 or보다 먼저 → (A and B) or C
채점 방식2000·1900·2024·2023 등 여러 연도로 자동 검사

✍️ 코드 작성 & 실행


  

  
💡 정답 코드 보기
    
def leap(year):
    if year % 4 == 0 and year % 100 != 0 or year % 400 == 0:
        return "윤년"
    else:
        return "평년"

    
2000년은 400의 배수라 윤년, 1900년은 100의 배수지만 400의 배수가 아니라 평년이에요.

  







  
02. 가위바위보 승부 판정 (중첩 if)10주 · 선택

  
📄 문제 : judge(player, computer)가 "비김" / "이김"(player 승) / "짐" 중 하나를 반환하게 하세요. (위 순서도 그대로)

  
▸ player가 이기는 3조합 : (가위→보) · (바위→가위) · (보→바위)

  
핵심 개념

  
개념설명

    
중첩 if마름모(조건) 안에 또 마름모 → else: 안의 if

    
복합 조건 or이기는 3가지 경우를 or로 한 번에

    
문자열 비교player == "가위"

  

  
✍️ 코드 작성 & 실행

  

    
    

    

    

  

  
💡 정답 코드 보기
    
def judge(player, computer):
    if player == computer:
        return "비김"
    else:
        if (player == "가위" and computer == "보") or \
           (player == "바위" and computer == "가위") or \
           (player == "보" and computer == "바위"):
            return "이김"
        else:
            return "짐"

    
9가지 경우(비김3·이김3·짐3)를 모두 자동 검사합니다.

출력 결과 (n=5)

  
*
**
***
****
*****

  
핵심 개념

  
개념설명

    
중첩 for바깥 for=줄, 안쪽 for=별 개수

    
print(end="")줄바꿈 없이 이어서 출력

✍️ 코드 작성 & 실행

05. 팩토리얼 (재귀 함수)13주 · 함수

  
📄 문제 : n! 을 재귀로 계산하는 factorial(n)을 작성하세요. (n ≥ 1)

  
▸ factorial(5) = 5 × 4 × 3 × 2 × 1 = 120

  
핵심 개념

  
개념설명

    
재귀(recursion)함수가 자기 자신을 호출

    
종료조건n == 1 이면 1 반환 (무한재귀 방지)

    
재귀식n * factorial(n - 1)

  

  
✍️ 코드 작성 & 실행

  

    
    

    

    

  

  
💡 정답 코드 보기
    
def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)

    
채점은 1!~8! 을 호출해 값이 맞는지 확인합니다.

  







  
06. Student 클래스 (성적 합·평균)14주 · 클래스

  
📄 문제 : 학생의 4과목 점수를 저장하고 총점·평균을 구하는 Student 클래스를 완성하세요.

  
▸ __init__(self, name, korean, math, english, science) · get_sum() · get_average()

  
핵심 개념

  
개념설명

    
__init__ / self객체 생성 시 속성 초기화, self.korean = korean

    
메서드객체에 속한 함수 — 합/평균 계산

    
메서드 재사용get_average는 self.get_sum()/4

  

  
✍️ 코드 작성 & 실행

  

    
    

    

    

  

  
💡 정답 코드 보기
    
class Student:
    def __init__(self, name, korean, math, english, science):
        self.name = name
        self.korean = korean
        self.math = math
        self.english = english
        self.science = science

    def get_sum(self):
        return self.korean + self.math + self.english + self.science

    def get_average(self):
        return self.get_sum() / 4

    
채점은 Student("홍길동", 100, 80, 90, 70) 로 합 340·평균 85.0 이 나오는지 확인합니다.
