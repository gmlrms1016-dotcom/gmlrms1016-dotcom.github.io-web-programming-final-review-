1. 웹 기초 개념1주차

① 인터넷 vs 웹(WWW)

인터넷 = 전 세계 컴퓨터를 연결한 '네트워크'. 웹(WWW) = 그 위에서 돌아가는 '서비스' 중 하나(이메일·FTP도 인터넷 서비스). 웹은 팀 버너스리가 발명, 표준은 W3C가 정합니다.

② 웹의 3대 핵심 규약

용어뜻
HTML웹 페이지를 기술하는 마크업 언어(내용)
URL인터넷 자원의 '주소'
HTTP서버↔브라우저가 문서를 주고받는 '전송 규약'

③ 웹페이지를 만드는 3대 기술 (역할 = 시험 단골!)

기술역할비유
HTML구조·뼈대(내용)집의 뼈대
CSS디자인·스타일(모양)페인트·장식
JavaScript동작·기능(움직임)전기·스위치

④ 태그(tag) 와 요소(element)

태그는 <...> 안의 단어. 요소는 <시작태그> + 내용 + </종료태그> 전체입니다.

예) <h1>나의 홈페이지</h1> → h1은 태그, 이 줄 전체는 하나의 요소. <img>·<br>처럼 닫는 태그가 없는 빈 태그도 있습니다.

HTML5는 최신 표준으로 video·audio(멀티미디어), 시맨틱 태그 등을 추가했습니다. 작성은 메모장이나 VS Code(Live Server 확장)로 합니다.

11. CSS 적용 3가지 방법6주차

스타일을 HTML에 입히는 방법은 3가지. 보통 ③ 외부 방식(이 문서!)을 가장 권장합니다.

방법위치특징
① 인라인태그 안 style="..."그 태그 1개만
② 내부head 안 <style>그 문서 전체
③ 외부.css 파일 + link여러 문서 공용(권장)

① 인라인 방식 — 이 문단만 빨강·굵게 (style 속성)

<!-- ① 인라인: 태그에 직접 (style 속성) -->
<p style="color:red;">빨강</p>

<!-- ② 내부: head 안에 모아서 -->
<head>
  <style>
    p { color: red; }
  </style>
</head>

<!-- ③ 외부: 파일 분리(가장 권장) -->
<link rel="stylesheet" href="style.css">  <!-- HTML쪽 -->
/* style.css 파일 안 */  p { color: red; }       /* CSS쪽 */

12. CSS 문법 구조 & 주석6주차

CSS 한 덩어리는 선택자 { 속성: 값; } 형태입니다. 무엇을(선택자) 어떻게(속성:값) 꾸밀지 정합니다.

선택자 {            /* 누구를 꾸밀지 (예: p, .box, #title) */
  속성: 값;         /* 무엇을 어떻게 (예: color: red;) */
  속성: 값;         /* 줄 끝마다 세미콜론(;) 필수! */
}                   /* { } 중괄호로 묶음 */

/* 실제 예시 */
h1 {                /* 모든 h1을 */
  color: blue;      /* 글자색 파랑 */
  font-size: 30px;  /* 크기 30px */
}                   /* /* ... */ 는 CSS 주석(설명, 화면에 안 보임) */
꼭 외울 것 : ① 콜론(:)으로 속성과 값 구분 · ② 세미콜론(;)으로 줄 끝 마무리 · ③ 중괄호 { }로 묶기 · ④ 주석은 /* ... */. (HTML 주석 <!-- --> 와 다름!)

13. 선택자 총정리6주차

'누구를 꾸밀지' 고르는 방법. #=id(1개), .=class(여러개), *=전체.

선택자고르는 대상
*모든 요소(전체)
p모든 <p> 태그(태그 선택자)
#titleid="title"인 요소 1개
.boxclass="box"인 모든 요소
div pdiv 안의 모든 p(후손)
h1, h2h1과 h2 동시에(그룹)
a:hover마우스 올린 a(가상클래스)
input[type=text]type이 text인 input(속성)

이 문단은 id로 골라 빨강 처리한 예시입니다.

← 이 링크에 마우스를 올리면 색이 변해요(:hover, CSS파일) 
* { margin: 0; }            /* 전체 선택 */
p { color: gray; }          /* 태그 선택 */
#title { font-size: 30px; } /* id 선택 (#, 한 개) */
.box { border: 1px solid; } /* class 선택 (점, 여러 개) */
div p { color: blue; }      /* 후손: div 안 모든 p */
h1, h2 { color: green; }    /* 그룹: 여러 개 한꺼번에 */

17. 박스 모델 (가장 중요!)10주차

속성의미

padding테두리 안쪽 여백
border테두리

box-sizingborder-box=테두리 포함해 width 계산

div {
  width: 200px;             /* 내용 너비 */

  border: 3px solid green;  /* 테두리: 두께 종류 색 */
  margin: 14px;             /* 바깥 여백(요소 사이) */
  box-sizing: border-box;   /* 테두리 포함해 너비 계산 */
}

/* 한 줄 4값: margin: 10px 20px 10px 20px; (상 우 하 좌) */

20. 위치(position) & z-index10주차

float는 요소를 왼쪽/오른쪽으로 띄워 옆으로 흐르게 합니다(예전 방식 2단 레이아웃). 뒤 요소가 딸려오면 clear로 끊습니다.

속성의미

22. flex 레이아웃9·13주차

부모에 display:flex를 주면 자식들이 한 줄로 정렬됩니다. 가로/세로 정렬을 쉽게 맞출 수 있어 메뉴·카드 배치에 많이 씁니다.

속성(부모에)의미
display: flexflex 켜기(자식 가로 정렬)
justify-content가로 정렬: flex-start·center·flex-end·space-between·space-around
align-items세로 정렬: flex-start·center·flex-end
flex-directionrow(가로·기본) / column(세로)

gap자식 사이 간격
A B C 
.menu {
  display: flex;                  /* 자식을 가로로 나란히 */

}

23. grid 레이아웃13주차

부모에 display:grid를 주고 grid-template-columns로 '몇 칸(열)'인지 정합니다. 격자(행·열) 배치에 강합니다.

속성의미

grid-template-columns열 개수·너비 지정

gap칸 사이 간격
1 2 3 4 5 6

  display: grid;                        /* 격자 모드 */

24. 전환 · 변형11주차

transform은 크기·회전·이동을 주고, transition은 그 변화를 '천천히' 일어나게 합니다. (아래 박스에 마우스를 올려보세요!)

속성/함수의미
transition: 0.3s0.3초에 걸쳐 부드럽게 변화
transform: scale(1.1)1.1배 확대
transform: rotate(45deg)45도 회전
transform: translateX(20px)오른쪽 20px 이동
마우스 올리면 1.1배 확대
.box {
  transition: transform 0.3s;  /* 0.3초 동안 부드럽게 */
}
.box:hover {                   /* 마우스를 올리면 */
  transform: scale(1.1);       /* 1.1배 확대 */
  /* rotate(45deg) 회전, translateX(20px) 이동도 가능 */
}

25. 애니메이션11주차

@keyframes로 동작 단계(0%~100%)를 정의하고, animation 속성으로 요소에 실행합니다. (마우스 없이도 저절로 움직임)

속성의미
@keyframes 이름0%~100% 동작 단계 정의
animation-name실행할 @keyframes 이름
animation-duration한 번 도는 시간(3s)
animation-iteration-count반복 횟수(infinite=무한)
3초마다 색이 바뀌어요(무한 반복)
@keyframes colorMove {           /* ① 동작 단계 정의 */
  0%   { background: green; }     /* 시작 */
  50%  { background: orange; }    /* 중간 */
  100% { background: green; }     /* 끝 */
}
.box {                           /* ② 요소에 실행 */
  animation-name: colorMove;          /* 위 이름 연결 */
  animation-duration: 3s;             /* 3초 동안 */
  animation-iteration-count: infinite;/* 무한 반복 */
}

26. 반응형 웹13주차

화면 크기(PC·태블릿·폰)에 따라 다른 스타일을 적용해, 어떤 기기에서도 잘 보이게 만듭니다.

요소역할
viewport meta(필수) 폭을 기기에 맞춤 — head에
@media (max-width:700px)폭 700px 이하일 때만 적용
가변 단위 %, em, rem고정 px 대신 비율로 → 늘었다 줄었다
📱 지금 브라우저 창을 좌우로 좁혀 보세요! 위 23번 grid 격자가 폭이 좁아지면 1줄로 바뀝니다. (이 페이지 CSS에 @media (max-width:700px)가 들어 있어요.)
